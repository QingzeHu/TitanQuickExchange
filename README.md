# TQE — Quick Stack Exchange

A WoW Classic quality-of-life addon that streamlines **merchant and trainer exchanges** into intuitive stack-based interactions.

## Features

- **Right-click once to exchange 1 full inventory stack**
- **Optional bulk mode** using player-entered target counts (all actions player-initiated)
- **Hover tooltips show true Classic stack semantics**, including:
  - Item name
  - Bag max stack size
  - Merchant minimum exchange unit
  - Estimated result based on Classic server rules
- **UI panel toggle (default OFF)** to enable stack or bulk mode
- **GitHub Release ZIP provided for direct download and easy installation**
- Fully compliant with Blizzard AddOn API (no memory or hardware input simulation)

## Installation

1. Download the addon ZIP from **GitHub Releases**
2. Extract into: `World of Warcraft_classic_titan_\Interface\AddOns\`
3. Enable **TQE** in the AddOns list
4. Reload UI in game: `/reload`

# TQE — 快速堆叠兑换（中文说明）

一个魔兽世界怀旧服 QoL 插件，将 **职业训练师和商人的兑换交互** 变成符合直觉的堆叠式操作。

## 功能

- **右键一次兑换 1 组背包允许的最大堆叠**
- **高级批量模式** 由玩家填写目标总数量触发（不自动挂机）
- 悬停 Tooltip 显示真正的怀旧服堆叠语义，包括：
- 物品名称
- 背包最大堆叠上限（堆叠组数）
- 商人最小兑换单位
- 基于怀旧服规则预测的最终获得结果
- UI 开关（默认关闭）可切换堆叠/批量模式
- GitHub Release 提供 ZIP 直接下载、安装更简单
- 完全符合暴雪 AddOn API 规范（不改内存、不模拟硬件）

## 安装方式

1. 在 GitHub **Releases** 下载插件 ZIP
2. 解压到正确目录：`World of Warcraft_classic_titan_\Interface\AddOns\`
3. 在插件列表启用 **TQE**
4. 进入游戏后输入：`/reload`
